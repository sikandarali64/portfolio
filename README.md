# Portfolio Website

A personal portfolio website showcasing projects, skills, and achievements in Computer System Engineering. Built using HTML, CSS, and JavaScript with a modern, responsive design.

## 🎯 About

This is my professional portfolio website where I showcase:
- My projects and work
- Technical skills and expertise
- Professional experience
- Contact information
- Resume and qualifications

## 🛠️ Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript
- **Styling**: Responsive CSS Grid & Flexbox
- **Design**: Modern and Professional UI/UX
- **Platform**: Static Website
